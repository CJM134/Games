# 1024组合游戏

一个简单的1024数字组合游戏，类似2048但目标是合成1024。

## 游戏规则

- 在4x4的网格中滑动方块
- 相同数字的方块碰撞后会合并成它们的和
- 每次移动后会根据分数难度生成新方块
- 目标是合成1024
- 当网格填满且无法移动时游戏结束

## 难度系统

游戏会根据你的分数自动调整难度：

- **简单模式（0-299分）**：每次移动后添加1个方块（2或4）
- **中等模式（300-499分）**：每次移动后添加2个方块（2或4）
- **困难模式（500分以上）**：每次移动后添加2个方块（4或8）

分数越高，游戏越具挑战性！

## 操作方式

- 上下左右滑动屏幕来移动方块
- 点击"重新开始"按钮重新开始游戏
- 点击"查看记录"查看历史成绩

## 功能特性

- ✨ 自动保存游戏记录
- 🏆 最高分排行榜
- 📊 历史成绩查看
- 🎮 动态难度调整
- 💾 数据持久化存储

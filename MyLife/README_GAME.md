# 中国式家长 - 简化版

## 游戏简介
这是一款模拟养成类游戏，灵感来自《中国式家长》。玩家扮演一个从小学到高中的学生，通过选择不同的行动来培养各项属性，最终获得毕业评价。

## 游戏玩法

### 核心机制
1. **属性系统**
   - 学习：影响考试成绩
   - 体育：身体素质
   - 艺术：艺术修养
   - 社交：人际关系
   - 压力：压力过高会影响成绩

2. **行动选择**
   每个学期有10次行动机会，可以选择：
   - 📚 学习：+8学习，+5压力
   - ⚽ 运动：+8体育，-3压力
   - 🎨 艺术：+8艺术，-2压力
   - 👥 社交：+8社交，-4压力
   - 😴 休息：-15压力
   - 📖 补习班：+12学习，+10压力（高收益高压力）

3. **随机事件**
   在行动过程中可能触发各种事件：
   - 考试成功（需要学习≥50）
   - 生病了
   - 朋友帮助（需要社交≥40）
   - 家长施压（学习<40时）
   - 才艺表演（艺术≥40时）
   - 运动会获奖（体育≥40时）

4. **学期评价**
   每个学期结束后会根据属性计算成绩：
   - S级（优秀）：300分以上
   - A级（良好）：250-299分
   - B级（中等）：200-249分
   - C级（及格）：150-199分
   - D级（不及格）：150分以下

5. **游戏流程**
   - 从小学一年级开始
   - 共12个学期（小学6年+初中3年+高中3年）
   - 每学期10次行动机会
   - 学期结束查看成绩
   - 高中毕业后查看最终评价

## 游戏策略提示

1. **平衡发展**：不要只关注学习，其他属性也很重要
2. **压力管理**：压力超过70会影响成绩，记得适时休息
3. **触发事件**：提升特定属性可以触发有利事件
4. **补习班使用**：在压力较低时使用补习班效果最好
5. **社交投资**：高社交可以触发朋友帮助事件

## 技术实现

### 项目结构
```
js/
├── config.js              # 游戏配置（行动卡片、事件等）
├── gamedata.js            # 游戏数据管理
├── main.js                # 游戏主循环
├── ui/
│   ├── gameui.js          # 游戏UI管理
│   ├── actioncard.js      # 行动卡片组件
│   └── eventpopup.js      # 事件弹窗组件
└── event/
    └── eventmanager.js    # 事件管理器
```

### 核心类说明

**GameData** - 游戏数据管理
- 管理学期进度
- 管理角色属性
- 计算成绩评价

**GameUI** - 游戏界面
- 显示顶部信息栏
- 显示属性面板
- 管理行动卡片
- 显示学期结束/游戏结束界面

**ActionCard** - 行动卡片
- 卡片点击交互
- 卡片动画效果

**EventPopup** - 事件弹窗
- 显示随机事件
- 事件效果展示

**EventManager** - 事件管理
- 触发随机事件
- 应用事件效果

## 扩展建议

如果想进一步完善游戏，可以考虑：

1. **更多事件**：添加更丰富的随机事件
2. **成就系统**：完成特定条件解锁成就
3. **选择分支**：某些事件提供多个选择
4. **特长系统**：专精某个属性可以解锁特殊能力
5. **家长互动**：添加与父母的对话系统
6. **存档系统**：保存游戏进度
7. **音效音乐**：添加背景音乐和音效
8. **动画效果**：更丰富的UI动画
9. **数据统计**：记录历史数据，生成图表
10. **多结局**：根据最终属性触发不同结局

## 开发说明

这个游戏是基于微信小游戏框架开发的，使用Canvas进行渲染。代码结构清晰，易于扩展和修改。

主要使用的技术：
- Canvas 2D API
- 事件驱动架构（TinyEmitter）
- 面向对象设计
- 游戏循环（requestAnimationFrame）
